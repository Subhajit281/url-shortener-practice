# URL Shortener (Backend Practice Project)

This is a beginner-level backend practice project built while learning Node.js and Express.

## Features
- Shorten long URLs
- Redirect to original URLs
- REST API based design
- MVC architecture
- Centralized error handling

## Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose

## Project Structure
- controllers – request handling logic
- routes – API routes
- models – database schemas
- middlewares – error handling
- utils – custom error classes

## Purpose
This project was created as a learning exercise to understand backend concepts such as routing,
controllers, models, middleware, and error handling.

## Future Improvements
- Authentication
- Rate limiting
- URL analytics
